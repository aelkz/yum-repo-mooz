# yum-repo-mooz

A yum repo maintained by [@moozhub](https://github.com/moozhub)

### Installing

CentOS / RHEL

```
curl -L https://raw.githubusercontent.com/moozhub/yum-repo-mooz/master/mooz-centos.repo > /etc/yum.repos.d/mooz-centos.repo
```

### Install packages

```
yum clean all
yum install kodi
```
